Node.js Application Deployment using Ansible

Overview This project automates the end-to-end deployment of a Node.js application using Ansible.
It eliminates manual setup by provisioning the server, building the application, and managing the service.


Features

🔧 Automated Node.js installation (manual binary setup)
📦 Application deployment from Git repository
📁 Dependency installation using npm
🔁 Process management using systemd
🔄 Idempotent Ansible playbook
🚀 One-command deployment

How to Run

Installs Node.js manually using tar package
Clones application code from Git
Installs dependencies using npm
Creates systemd service
Starts and enables the application
